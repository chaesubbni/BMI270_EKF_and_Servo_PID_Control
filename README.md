# BMI270 EKF & Servo PID Stabilization Control

BMI270 IMU 센서와 **확장 칼만 필터(EKF)**를 활용하여 고정밀 자세(Roll)를 추정하고, 저주파 필터가 적용된 **PID 제어**를 통해 서보 모터를 구동하여 시스템을 수평으로 유지하는 프로젝트입니다. 
드론의 수평 제어를 하기 전 PID를 먼저 연습하기 위해 미니 프로젝트 형식으로 진행했습니다.

---

## 주요 기능 및 특징

* **5차원 확장 칼만 필터 (EKF):** `ArduinoEigen` 라이브러리를 기반으로 오일러 각도의 비선형 방정식과 자코비안 행렬을 매 루프마다 연산하여 Roll, Pitch 및 3축 자이로 센서의 편향(Bias)을 실시간으로 추정합니다.
* **자동 영점 및 노이즈 센싱:** 초기 부팅 시 300개의 샘플을 수집하여 센서의 평균 오프셋을 제거하고, 분산을 계산해 EKF의 시스템 노이즈 공분산 행렬(Q)과 측정 노이즈 공분산 행렬(R)을 동적으로 설정합니다.
* **실시간 PID & LPF 튜닝:** 시리얼 모니터 키 입력(`P/p`, `I/i`, `D/d`, `A/a`)을 통해 코드를 다시 업로드하지 않고도 PID Gain 값과 미분항 LPF 계수를 실시간으로 미세 조정할 수 있습니다.
* **동적 제어 주기 동기화:** 센서 데이터 업데이트 주기(12ms)와 서보 PID 제어 주기(50ms)가 독립적으로 작동 구동되며, 실제 소요 시간(`pid_dt`)을 동적으로 계산하여 정확한 미적분 제어를 수행합니다.

## 느낀점

본격적인 드론 수평 PID 제어에 들어가기에 앞서, 사전 빌드업이자 개념 검증 단계로 시도해 본 미니 프로젝트였습니다.
실제로 모터나 프레임을 손으로 비틀고 움직일 때마다 센서가 이를 즉각적으로 감지하고, 서보 모터가 "지잉- 지잉-" 소리를 내며 알아서 척척 수평을 맞춰가는 모습을 보니 정말 신기했습니다. 제어 주기에 맞춰 빠르고 촘촘하게 반응할 때 발생하는 특유의 모터 진동과 구동음 덕분에 하드웨어 제어의 쾌감을 제대로 느낄 수 있었던 즐거운 프로젝트였습니다.
이번에 EKF와 PID의 뼈대를 확실히 다져둔 덕분에, 향후 진행할 드론의 멀티 축 수평 제어도 자신 있게 도전할 수 있을 것 같습니다!
